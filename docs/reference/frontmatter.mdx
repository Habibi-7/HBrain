## Fields

| Field | Required | Value |
|-------|----------|-------|
| `id` | Yes | ULID — must match the filename prefix |
| `schema` | Yes | `1` |
| `type` | Yes | `note` \| `task` \| `decision` \| `fact` \| `link` |
| `created_at` | Yes | ISO-8601 UTC timestamp |
| `source` | Yes | `agent` |
| `agent` | Yes | Agent name, e.g. `claude-code`, `cowork`, `manus` |
| `tags` | No | YAML list, lowercase, 1–3 relevant tags |
| `links` | No | YAML list of related event ULIDs |
| `status` | Tasks only | `open` \| `done` \| `blocked` \| `cancelled` |

## Example

```markdown
---
id: 01JVMY7QXR8KF3DNQJ5CGPXG9S
schema: 1
type: decision
created_at: 2026-05-04T14:32:11Z
source: agent
agent: claude-code
tags: [backend, database]
links: [01JVMX3AB...]
status: open
---

Event body in plain markdown.
```

## Immutable fields

`id` and `created_at` **never change** once written. You can edit the body,
tags, links, and status. Changing `id` or `created_at` breaks the system.

## Generating a ULID

```bash
python3 -c "from ulid import ULID; print(ULID())"
```

If `ulid` isn't available, any 26-char time-sortable ID works. The first 10
chars should encode the current timestamp in Crockford Base32.
