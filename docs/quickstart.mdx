## Install

### Claude Code / Cowork

```bash
cp -r skill/SKILL.md ~/.claude/skills/brain/SKILL.md
```

Then copy templates to your vault (after creating one):

```bash
mkdir -p ~/brain/.brain/templates
cp skill/templates/* ~/brain/.brain/templates/
```

### Other agents

The skill is a self-contained markdown document. Install it however your agent platform supports loading skills.

## Set a vault path

Tell your agent where to store your vault, or set `$BRAIN_DIR`:

```bash
export BRAIN_DIR=~/brain
```

If `$BRAIN_DIR` isn't set, the agent will ask you once and remember the path.

## Capture your first event

Just talk to your agent:

> "Remember that I decided to use Postgres for its native JSON support."

The agent creates a file like:

```
~/brain/events/2026/05/04/01JVMY7QX-decided-postgres.md
```

With contents:

```markdown
---
id: 01JVMY7QXR8KF3DNQJ5CGPXG9S
schema: 1
type: decision
created_at: 2026-05-04T14:32:11Z
source: agent
agent: claude-code
tags: [backend, database]
links: []
---

Decided to use Postgres for its native JSON support.
```

The agent confirms: "Captured decision · `01JVM...`"

## Query your events

> "What did I decide about the database?"

> "Show my week"

> "List my open tasks"

The agent reads your vault files and responds.
